Оригинал
block
блок
noun
имя существительное
A unit of page output that renders some distinctive content – a piece of information, a user interface element – anything visually tangible for the end-user. Blocks are implemented and provided by modules. Blocks use templates to generate HTML. Examples of blocks include a category list, a mini cart, product tags, and product listing.
Единица вывода страницы, которая отображает некоторый отличительный контент - часть информации, элемент пользовательского интерфейса - все, что визуально осязаемо для конечного пользователя. Блоки реализуются и предоставляются модулями. Блоки используют шаблоны для генерации HTML. Примеры блоков включают список категорий, мини-корзину, теги продуктов и список продуктов.
Dynamic blocks provide content based on logic, such as price rules.
Динамические блоки предоставляют контент, основанный на логике, например ценовых правилах.
Page Builder expands on the interactivity and creation of blocks and dynamic blocks.
Page Builder расширяет возможности интерактивности и создания блоков и динамических блоков.
