Magento 2 Module Development - Simple Hello World Module
Разработка модуля для Magento 2 - простой модуль Hello World
Magento 2 Module development or Magento 2 Hello World trends is increase rapidly while Magento release official version. That why we - Mageplaza - are wring about a topic that introduces how to create a simple Module.
Разработка модуля Magento 2 или Magento 2 Hello World стремительно набирают обороты, в то время как Magento выпускает официальную версию. Вот почему мы - Mageplaza - обсуждаем тему, которая знакомит с тем, как создать простой модуль.
As you know, the module is a directory that contains blocks, controllers, models, helper, etc - that are related to a specific business feature. The etc folder contains admin ACL, admin menu, configuration . In Magento 2, modules will be live in app/code directory of a Magento installation, with this format: app/code//. Now we will follow this steps to create a simple module which work on Magento 2 and display Hello World.
Как вы знаете, модуль - это каталог, содержащий блоки, контроллеры, модели, помощники и т. Д., Относящиеся к определенной бизнес-функции. Папка etc содержит ACL администратора, меню администратора, конфигурацию. В Magento 2 модули будут находиться в каталоге app / code установки Magento в следующем формате: app / code / / . Теперь мы выполним следующие шаги, чтобы создать простой модуль, который будет работать в Magento 2 и отображать Hello World.

Create Hello World module for Magento 2
Создать модуль Hello World для Magento 2
To create Hello World module, you need to complete the following high-level steps:
Чтобы создать модуль Hello World, вам необходимо выполнить следующие высокоуровневые шаги:
Step 1: Create the folder of Hello World module
Шаг 1: Создайте папку модуля Hello World
Step 2: Create etc/module.xml file
Шаг 2. Создайте файл etc / module.xml
Step 3: Create etc/registration.php file
Шаг 3. Создайте файл etc / registration.php
Step 4: Enable the module
Шаг 4. Включите модуль
Step 1: Create the folder of Hello World module
Шаг 1: Создайте папку модуля Hello World
Name of the module is defined as VendorName_ModuleName. First part is name of the vendor and last part is name of the module: For example: Magento_HelloWorld, Mageplaza_PdfInvoice. Focus on following guide to create the folders:
Имя модуля определяется как VendorName_ModuleName. Первая часть - это имя производителя, а последняя часть - это имя модуля: Например: Magento_HelloWorld, Mageplaza_PdfInvoice. Сосредоточьтесь на следующем руководстве по созданию папок:
  app/code/Mageplaza/HelloWorld
Step 2: Create etc/module.xml file.
Шаг 2: Создайте файл etc/module.xml.
Then, it is necessary to create etc folder and add the module.xml file
Затем необходимо создать папку etc и добавить файл module.xml
  app/code/Mageplaza/HelloWorld/etc/module.xml
Contents would be:
Содержание будет:
<?xml version="1.0"?>
<config xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xsi:noNamespaceSchemaLocation="urn:magento:framework:Module/etc/module.xsd">
    <module name="Mageplaza_HelloWorld" setup_version="1.0.0">
    </module>
</config>
=======================================================================
Step 3: Create etc/registration.php file
Шаг 3. Создайте файл etc / registration.php
In this step, we will add registration.php as following guide:
На этом этапе мы добавим registration.php в качестве следующего руководства:
  app/code/Mageplaza/HelloWorld/registration.php
Contents would be:
Содержание будет:
<?php
\Magento\Framework\Component\ComponentRegistrar::register(
	\Magento\Framework\Component\ComponentRegistrar::MODULE,
	'Mageplaza_HelloWorld',
	__DIR__
);
Step 4: Enable the module
Шаг 4. Включите модуль
Finish the step 3, we have already created the HelloWorld module. And we will enable this module in this step
Завершите шаг 3, мы уже создали модуль HelloWorld. И мы включим этот модуль на этом шаге
After create the module if you run the command as:
После создания модуля, если вы запустите команду как:
  php bin/magento module:status
You should see the module is disable now:
Вы должны увидеть, что модуль отключен:
List of disabled modules: Mageplaza_HelloWorld
Список отключенных модулей: Mageplaza_HelloWorld
Follow exact guide to enable the module right now, let run the command as:
Следуйте точному руководству, чтобы включить модуль прямо сейчас, запустите команду как:
  php bin/magento module:enable Mageplaza_HelloWorld
Or other way, you can access the file:
Или другим способом вы можете получить доступ к файлу:
  app/etc/config.php
You will see a long list of modules there, just add your module as well
Вы увидите там длинный список модулей, просто добавьте свой модуль
  ...
  'Mageplaza_HelloWorld' => 1, 
  ...
Your module should be available now.
Ваш модуль должен быть доступен сейчас.
After this step, when you open your website in browser you will get an error saying
После этого шага, когда вы откроете свой веб-сайт в браузере, вы получите сообщение об ошибке
Please upgrade your database: Run bin/magento setup:upgrade from the Magento root directory.
Пожалуйста, обновите свою базу данных: запустите bin/magento setup:upgrade из корневого каталога Magento.
Let run the command:
Запустим команду:
  php bin/magento setup:upgrade
After complete, when you open your website in browser you will see the layout of the website is broken.
После завершения, когда вы откроете свой веб-сайт в браузере, вы увидите, что макет веб-сайта нарушен.
Please run the deloy command line to fix it.
Пожалуйста, запустите командную строку deloy, чтобы исправить это.
php bin/magento setup:static-content:deploy
After deploy completed, you can also see your module from backend at System Configuration -> Advanced -> Disable Modules Output.
После завершения развертывания вы также можете увидеть свой модуль из серверной части в Конфигурация системы -> Дополнительно -> Отключить вывод модулей.
Now, we will create a controller to test module.
Теперь мы создадим контроллер для тестирования модуля.
Before create a controller, we will create a route for HelloWorld module.
Перед созданием контроллера мы создадим маршрут для модуля HelloWorld.
Route’s in magento are divided into 3 parts: Route frontname, controller and action as following example:
Маршрут в magento разделен на 3 части: имя маршрута, контроллер и действие, как показано в следующем примере:
http://mageplaza.com/index.php/frontname/controller/action
To add route, it is necessary to create routes.xml file
Для добавления маршрута необходимо создать файл routes.xml.

app/code/Mageplaza/HelloWorld/etc/frontend/routes.xml

since this is a frontend route, we added it in frontend/ folder else we need to add it to adminhtml/folder
поскольку это интерфейсный маршрут, мы добавили его в папку frontend /, иначе нам нужно добавить его в папку adminhtml /
Content would be:
Контент будет:
<?xml version="1.0" ?>
<config xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xsi:noNamespaceSchemaLocation="urn:magento:framework:App/etc/routes.xsd">
    <router id="standard">
        <route frontName="helloworld" id="helloworld">
            <module name="Mageplaza_HelloWorld"/>
        </route>
    </router>
</config>
After defining the first part of the route, the URL will be displayed as:
После определения первой части маршрута URL-адрес будет отображаться как:
http:///helloworld/*
http:///helloworld/*
E.g: http://localhost/helloworld/*
Например: http://localhost/helloworld/*
Then, we will continue to create the controller and action
Затем мы продолжим создание контроллера и действия
The folder and file you need to create is:
Папка и файл, которые вам нужно создать:
app/code/Mageplaza/HelloWorld/Controller/Index/Test.php
app/code/Mageplaza/HelloWorld/Controller/Index/Test.php
Contents would be:
Содержание будет:
<?php
namespace Mageplaza\HelloWorld\Controller\Index;

class Test extends \Magento\Framework\App\Action\Action
{
	protected $_pageFactory;

	public function __construct(
		\Magento\Framework\App\Action\Context $context,
		\Magento\Framework\View\Result\PageFactory $pageFactory)
	{
		$this->_pageFactory = $pageFactory;
		return parent::__construct($context);
	}

	public function execute()
	{
		echo "Hello World";
		exit;
	}
}
After completed, please run php bin/magento cache:clean to check result.
После завершения запустите php bin/magento cache:clean, чтобы проверить результат.
Your URL now should be as:
Теперь ваш URL-адрес должен быть таким:
http:///helloworld/index/test
http:///helloworld/index/test
After finish all steps, the output Hello World should be displayed in your browser when you open the URL. We hope our guide is very useful and effective for you. Any questions, feel free to leave a commend below. In the next tutorial, you will learn how to create a controller.
