Build
Строить

Building your component involves laying out the file structure, creating the necessary configuration files, building out any needed API interfaces and services, and adding any frontend parts needed for your component.
Создание вашего компонента включает в себя создание файловой структуры, создание необходимых файлов конфигурации, создание любых необходимых интерфейсов и служб API, а также добавление любых частей внешнего интерфейса, необходимых для вашего компонента.

Prerequisites
Предпосылки

Before you begin creating your new component, make sure that you have a working installation of Magento 2, and the Magento System Requirements.
Прежде чем приступить к созданию нового компонента, убедитесь, что у вас есть рабочая установка Magento 2 и системные требования Magento.
Also, Magento recommends that you disable caching while setting up the component file structure and adding configuration files.
Также Magento рекомендует отключить кеширование при настройке файловой структуры компонента и добавлении файлов конфигурации.

Ниже приводится подробная информация о процессе сборки компонентов:
